O que é o Instantfy?

Um aplicativo para simplificar a vida de quem precisa disponibilizar uma agenda para marcar qualquer horario.

Nós queremos tornar o agendamento de qualquer atividade o mais simples possível, facilitando tanto para quem quer buscar horarios disponíveis quanto para quem quer disponibilizar um serviço ou uma locação por exemplo.
